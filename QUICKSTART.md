# Быстрый старт

## 1. Установка зависимостей

```bash
npm install
```

## 2. Запуск в режиме разработки

```bash
npm run dev
```

Откроется браузер на http://localhost:3000

## 3. Что нужно изменить перед деплоем

### Обязательно:
- [ ] **Контакты в Footer.jsx** (телефон, email, telegram)
- [ ] **ФИО в About.jsx** (если нужно изменить)
- [ ] **Подключить формы к бэкенду** в Business.jsx и Course.jsx

### Рекомендуется:
- [ ] Добавить реальные изображения (если требуется)
- [ ] Настроить мета-теги в index.html
- [ ] Добавить Google Analytics или Яндекс.Метрику
- [ ] Проверить все тексты на актуальность

## 4. Сборка для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

## 5. Деплой

### Vercel (рекомендуется):
```bash
npm install -g vercel
vercel
```

### Netlify:
1. Загрузите папку `dist` на Netlify
2. Или подключите GitHub репозиторий

### Обычный хостинг:
1. Загрузите содержимое папки `dist/` на сервер
2. Настройте веб-сервер на отдачу index.html

## Структура файлов для редактирования

```
src/
├── components/
│   ├── Header.jsx       # Верхнее меню
│   ├── Hero.jsx         # Главный экран с кнопками
│   ├── About.jsx        # Информация об эксперте
│   ├── Business.jsx     # Услуги для бизнеса + форма
│   ├── Course.jsx       # Курс "яПрав!" + форма
│   └── Footer.jsx       # Контакты (ИЗМЕНИТЬ ЗДЕСЬ!)
└── styles/
    └── global.css       # Цвета и глобальные стили
```

## Основные команды

| Команда | Описание |
|---------|----------|
| `npm install` | Установка зависимостей |
| `npm run dev` | Запуск dev-сервера |
| `npm run build` | Сборка для продакшена |
| `npm run preview` | Предпросмотр prod-сборки |
