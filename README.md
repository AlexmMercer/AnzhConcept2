# ВоВсемПрав.ру - Лендинг правового досье-офиса

Лендинг для юридического сервиса, выполненный в концепции "папки-досье" с двумя направлениями:
- Правовое сопровождение бизнеса
- Курс "яПрав!" для частных лиц

## Особенности реализации

### Визуальная концепция
- Дизайн в стиле папки с документами
- Интерактивные закладки для навигации
- Анимация перелистывания страниц
- Визуальные "штампы" и "листы"

### Технологии
- React 18
- Vite
- CSS3 с анимациями
- Адаптивная верстка (desktop/tablet/mobile)

### Функционал
- Плавная прокрутка между секциями
- Интерактивные формы с валидацией
- Раскрывающиеся карточки услуг
- Анимация появления элементов при скролле
- Всплывающие подсказки для тем курса

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение откроется автоматически на http://localhost:3000

### Сборка для продакшена
```bash
npm run build
```

### Предварительный просмотр production сборки
```bash
npm run preview
```

## Секции сайта

### 1. Hero (#hero)
- Обложка папки с логотипом
- Две основные CTA-кнопки (бизнес и курс)
- Интерактивные закладки для навигации
- Визуал открытой папки справа

### 2. Лист 1. Эксперт (#about)
- Карточка эксперта с полями данных
- Визуальные "штампы" образования и квалификаций
- Блок "Почему ко мне возвращаются?"

### 3. Лист 2. Решения для бизнеса (#business)
- Таблица-карточки услуг с hover-эффектами
- Раскрывающиеся примеры кейсов
- Форма заявки на консультацию (sticky)

### 4. Лист 3. Курс "яПрав!" (#course)
- Описание курса и преимуществ
- Чек-лист из 5 тем с анимацией появления
- Всплывающие подсказки при наведении
- Форма записи на курс

### 5. Футер (#footer)
- Дизайн в виде кармана папки
- Контактная информация
- Две CTA-кнопки (дублирующие)
- Копирайт

## Шрифты

- **PT Serif** - заголовки (Google Fonts)
- **PT Sans** - основной текст (Google Fonts)
- **PT Sans Caption** - подписи и метки (Google Fonts)

## Анимации

- Появление Hero-блока при загрузке
- Анимация "перелистывания" листов при скролле
- Последовательное появление элементов чек-листа
- Hover-эффекты для карточек и кнопок
- Плавная прокрутка между якорями (smooth scroll)
- Shake-анимация для ошибок в формах

## Адаптивность

- **Desktop**: ≥1024px - двухколоночные макеты, sticky-формы
- **Tablet**: 768-1023px - адаптация колонок, вертикальный стек
- **Mobile**: ≤767px - одноколоночный макет, вертикальные закладки

## Формы

Обе формы (бизнес и курс) включают:
- Плавающие лейблы (floating labels)
- Валидацию с shake-эффектом
- Подсветку активных полей
- Чекбокс согласия на обработку данных
- Информационные подсказки

## Контакты для обновления

В файлах компонентов указаны placeholder-контакты:
- Телефон: Footer.jsx (строка с `+7 (XXX) XXX-XX-XX`)
- Email: Footer.jsx
- Telegram: Footer.jsx

Замените их на реальные контактные данные перед деплоем.
